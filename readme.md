#orquery v1.0 by 糸貫内科クリニック

日医標準レセプトソフトORCAのデータベースに接続して、患者番号から患者氏名等を所得し、標準出力に書き出します。
このプログラムの出力をリダイレクトして、他のアプリケーションで利用することを想定しています。
当院では、ラベル印刷アプリケーションや、他社システムとの連携アプリケーションの一部として用いております。


##使い方：
orquery.exe 患者番号 ORCAのIPアドレス (必要ならパスワード)


例：orquery.exe 1234 192.168.2.1


詳しくはソースコードを参照してください
